Manual Service Installation Required

The service must be installed manually since this web panel runs inside the service itself.

Please run the following command in your terminal, then refresh this page:

mkdir -p ~/.config/systemd/user && cp service/strike-connect.service ~/.config/systemd/user/ && systemctl --user daemon-reload && systemctl --user enable strike-connect && systemctl --user start strike-connect

After running the command, wait a few seconds and refresh this page to see the service status.
